# Session 2 (Friday) — Dictionaries & Sets

## What we're covering today

1. **[Module 1 — Dictionaries: Key-Value Mapping](./01-dictionaries.md)**
   Creating dicts, accessing values safely with `.get()`, iterating with `.items()`, nested dictionaries.
2. **[Module 2 — Sets: Unique Collections](./02-sets.md)**
   Creating sets, set operations (union, intersection, difference), deduplication, when to use sets vs lists.

## Before you start

Make sure you're comfortable with lists and tuples from Session 1 — dictionaries and sets build on those concepts.

## Runnable code for this session

All demo code lives in [`code/`](./code/). Every file is meant to be run, not just read — open a terminal in this folder, activate your venv, and run each one as you reach it in the handbook:

```bash
python code/demo_dict_basics.py
python code/demo_dict_methods.py
python code/demo_nested_dicts.py
python code/demo_dict_iteration.py
python code/demo_sets_basics.py
python code/demo_sets_operations.py
```
