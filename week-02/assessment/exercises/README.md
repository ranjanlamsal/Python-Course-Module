# Week 2 Exercises

These exercises are meant to be completed before you start the Contact Book project.

## Suggested order

1. [exercise_1_list_manipulation.py](./exercise_1_list_manipulation.py) — work with lists, sorting, and slicing
2. [exercise_2_tuple_unpacking.py](./exercise_2_tuple_unpacking.py) — practice tuples and unpacking
3. [exercise_3_contact_directory.py](./exercise_3_contact_directory.py) — build a small dictionary-based directory
4. [exercise_4_set_deduplication.py](./exercise_4_set_deduplication.py) — use sets to remove duplicates

## How to use them

- Open each file in your editor.
- Read the instructions in the docstring.
- Write your code where you see the `# TODO:` markers.
- Run each file from the terminal to test your solution.

If you finish all four, move on to the project spec in [../project/PROJECT_SPEC.md](../project/PROJECT_SPEC.md).
