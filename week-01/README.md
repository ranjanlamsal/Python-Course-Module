# Week 1 — The Absolute Basics & Control Flow

## Why this week matters

Everything else in this course — data structures, functions, file handling, OOP — is built out of four things: **variables, types, branching, and iteration.** If any of these four feel shaky by Wednesday night, they will actively block you in Week 2. Use that support now.

## Objectives — by Thursday night you should be able to:

- Explain, in your own words, why we use a virtual environment instead of installing packages globally.
- Create and activate a `venv`, and know how to tell it's active.
- Predict the output of a mixed-type arithmetic expression before running it.
- Explain the difference between `//` and `%`, without looking it up.
- Write a `while` loop with a controlled exit and a `for` loop over a `range()`.
- Read a stack trace far enough to identify *which line* and *what kind* of error occurred.
