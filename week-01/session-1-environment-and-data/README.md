## Before you start

Have VS Code and Python installed and ready to open a terminal. If you haven't installed either yet, Module 1 covers it first.

## Runnable code for this session

All demo code lives in [`code/`](./code/). Every file is meant to be run, not just read — open a terminal in this folder, activate your venv, and run each one as you reach it in the handbook:

```bash
python code/hello_world.py
python code/demo_variables_and_types.py
python code/demo_dynamic_typing.py
python code/demo_type_casting.py
```
