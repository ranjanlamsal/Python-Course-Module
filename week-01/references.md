# Week 1 — Deep Dive References

These are official, stable resources — safe to bookmark for the entire course, not just this week. When a handbook module says "check the docs," this is where to look.

## General / Course-Wide

- **Python official docs home** — `docs.python.org` — the root of everything below; if a specific link ever moves, start here and navigate in.
- **The Python Tutorial (official)** — `docs.python.org/3/tutorial/` — the canonical beginner walkthrough Python's own maintainers write and keep updated. Every module this week has a matching section here.
- **Real Python** — `realpython.com` — a large, well-regarded, actively maintained hub of Python tutorials aimed at exactly this level and beyond. Use the site's own search for whatever concept you need ("Real Python venv", "Real Python type casting", etc.) rather than a specific article link, since individual URLs there change over time.

## Module 1 — Environment Setup & Isolation

- **Installing Python** — docs.python.org's "Using Python" section (`docs.python.org/3/using/`), covering setup on Windows/macOS/Linux.
- **`venv` — Creation of virtual environments** — `docs.python.org/3/library/venv.html` — the official reference for exactly what we did in Module 1.
- **VS Code Python documentation** — `code.visualstudio.com/docs/languages/python` — the editor-side setup guide (extension install, interpreter selection) referenced from within VS Code's own docs.

## Module 2 — Data Fundamentals & Transformation

- **The Python Tutorial, "An Informal Introduction to Python"** — the tutorial section covering numbers, strings, and basic types — `docs.python.org/3/tutorial/introduction.html`.
- **Built-in Types reference** — `docs.python.org/3/library/stdtypes.html` — the exhaustive, authoritative list of what `int`, `float`, `str`, and `bool` can do.
- **Built-in Functions reference** (for `int()`, `str()`, `float()`, `type()`) — `docs.python.org/3/library/functions.html`.

## Module 3 — Operators & Branching Logic

- **The Python Tutorial, "More Control Flow Tools"** — `docs.python.org/3/tutorial/controlflow.html` — covers `if`/`elif`/`else` directly from the source.
- **Expressions reference (operators, including `//`, `%`, `**`)** — `docs.python.org/3/reference/expressions.html`.
- **Real Python** — search their site for "Python operators" and "Python if statements" for worked examples beyond the official docs.

## Module 4 — Iteration Essentials

- **The Python Tutorial, "More Control Flow Tools"** (same page as above) — also covers `while`, `for`, `break`, `continue`, `pass`, and `range()` directly beneath the conditionals section.
- **`range()` built-in reference** — `docs.python.org/3/library/stdtypes.html#range` — the exact, authoritative behavior of `range()`'s start/stop/step arguments.
- **Real Python** — search their site for "Python while loops" and "Python for loops" for additional worked walkthroughs.

## A note on using these independently

If a link above looks slightly different from what you find (Python's docs occasionally restructure), don't panic — use the site's own search bar with the concept name. The goal of this list is to point you at *trustworthy sources*, not to be a perfect permanent map. Official docs and Real Python are both safe defaults for the rest of this course whenever this repo doesn't cover something in enough depth.
