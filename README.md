# MIT Kathmandu — Accelerated Python Course

A 4-week, ground-up Python course for absolute beginners. 8 live sessions (Thu/Fri, 90 min each), heavy self-directed build time (Sat–Wed) between them. This repo is the single source of truth: handbooks, runnable demo code, and weekend assessments, organized by week and session so you can follow along live or work through it independently later.

## How this repo is organized

```
mit-python-course/
├── SCHEDULE.md                 ← the full 4-week plan (what's taught when, and why)
├── week-01/
│   ├── README.md                ← week overview, objectives, links to everything below
│   ├── session-1-.../           ← handbook modules + runnable demo .py files for Thu
│   ├── session-2-.../           ← handbook modules + runnable demo .py files for Fri
│   ├── references.md            ← curated official docs / further reading
│   └── assessment/               ← the weekend workbook: exercises, project spec, optional challenges, reflection log
├── week-02/  (added as the course progresses)
└── ...
```

Each week follows this same shape. Nothing here is graded — assessments exist so that the next session's 30-minute instructor-led review has real material to dissect.

## How to use this repo

- **If you're in class:** the handbook `.md` files are what's on the projector. Follow along, run the demo `.py` files yourself in your own terminal as we go.
- **If you're working solo over the weekend (or studying independently later):** read the handbook modules in order first, run every demo file, do the Self-Checks honestly before peeking at the answers, *then* open `assessment/` and work through it in the order it's laid out.
- **If you get stuck with no instructor around:** re-read the "⚠️ Common Pitfalls" box for the concept you're using — it exists specifically for this moment. Then check `references.md` for the official docs on that exact topic.
